# v0.7 Live Static Prototype

更新时间：2026-05-19

## 目标

基于用户确认，开发实时谈案 Copilot 静态原型。

确认项：

- 案由：劳动争议
- 中栏：客户 / 律师分角色对话流
- 法条：第一版使用模拟法条卡，不接真实检索
- 咨询结束后保留：咨询纪要、材料清单、方案摘要
- 法律意见书：保留为“模拟法律意见书格式样例”，明确未审核不可发送

## 原型路径

```text
prototype/index.html
```

## 页面结构

```text
左栏：咨询会话列表
中栏：实时转写对话流
右栏：案情要点 / 追问建议 / 模拟法条卡 / 方案草稿 / 结束后产物
```

## 演示脚本

劳动争议违法解除咨询：

1. 客户陈述入职时间、岗位、工资。
2. 客户陈述公司以绩效不达标口头解除。
3. 律师追问绩效考核和改进计划。
4. 客户说明已有劳动合同、工资流水、微信记录，缺完整考勤。
5. 客户提出赔偿金、年假工资、加班费诉求。

每段触发右侧更新：

- 案情要点
- 建议追问
- 模拟法条卡
- 方案草稿

## 交互修正

根据试用反馈，v0.7 原型调整为：

- 点击“开始咨询”后自动流式输出客户/律师对话。
- 不再需要点击“下一段”。
- 右侧随双方对话自动同步更新。
- 点击“结束咨询”后，右侧四个产物按钮点亮。
- 点击产物按钮后，在中间转写区追加展示对应内容预览。
