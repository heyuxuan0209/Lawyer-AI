# 劳动争议 MVP 输出模板

本文件汇总 MVP 第一版的 5 类输出，供产品原型和 Prompt 调用。

## 1. 咨询纪要

目标：留痕、复盘、后续生成法律意见书。

核心结构：

```text
一、咨询基本信息
二、客户主要诉求
三、事实经过
四、现有材料
五、律师咨询中的初步回应
六、待补充事项
七、后续安排
八、需律师复核的重点
```

## 2. 案情结构化表

目标：让律师快速看到劳动争议判断所需字段。

核心结构：

```text
一、当事人信息
二、劳动关系信息
三、争议类型初步归类
四、关键时间线
五、客户主张
六、争议焦点
七、现有证据
八、缺失信息
九、初步风险点
```

## 3. 待补材料清单

目标：转成客户可执行的补材料任务。

核心结构：

```text
一、必须补充材料
二、建议补充材料
三、如无法提供，需补充说明
四、客户发送说明
五、律师内部提示
```

## 4. 初步法律分析草稿

目标：作为律师内部分析底稿，不直接发客户。

核心结构：

```text
一、可能法律关系
二、可能争议焦点
三、客户可能主张
四、对客户有利的因素
五、对客户不利或不确定的因素
六、需要进一步核验的问题
七、初步处理路径
八、风险提示
```

## 5. 法律意见书草稿

目标：基于付费咨询形成可由律师修改确认的书面交付件。

核心结构：

```text
一、咨询背景
二、基本事实
三、咨询问题
四、初步法律分析
五、处理建议
六、待补材料
七、风险提示
八、律师确认区
```

## 产品展示建议

在三栏式 MVP 中：

- 中栏展示咨询处理过程：原始记录 -> 纪要 -> 分析 -> 意见书。
- 右栏展示可折叠交付件：结构化表、材料清单、意见书草稿。
- 每个输出块都标注状态：`AI 草稿`、`待律师确认`、`已确认`。
