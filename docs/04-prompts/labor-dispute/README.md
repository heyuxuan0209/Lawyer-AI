# 劳动争议 Prompt 目录

本目录用于沉淀劳动争议咨询助手的 Prompt 和输出模板。

计划拆分：

- `consultation-minutes.prompt.md`：咨询纪要生成
- `case-structure.prompt.md`：案情结构化
- `material-checklist.prompt.md`：待补材料清单
- `initial-legal-analysis.prompt.md`：初步法律分析
- `legal-opinion-draft.prompt.md`：法律意见书草稿
- `output-templates.md`：MVP 输出模板汇总

原则：

- 所有输出均为律师复核草稿。
- 不直接承诺胜诉率。
- 不伪造法条和案例。
- 法律依据不确定时标注“需进一步核验”。
