# MVP 评估标准

用于评估劳动争议咨询助手生成结果是否具备真实产品价值。

## 1. 咨询纪要

评估项：

- 是否覆盖客户核心诉求
- 是否保留关键事实
- 是否去除了无关口语噪音
- 是否适合留档

## 2. 案情结构化

评估项：

- 是否准确提取劳动关系信息
- 是否识别关键时间节点
- 是否识别争议焦点
- 是否区分现有证据和缺失证据

## 3. 待补材料清单

评估项：

- 是否列出必要材料
- 是否按重要性分层
- 是否能让客户看懂
- 是否避免过度索取无关材料

## 4. 初步法律分析

评估项：

- 是否只作为律师内部草稿
- 是否标注需核验问题
- 是否避免确定性胜诉承诺
- 是否提示不利因素和风险

## 5. 法律意见书草稿

评估项：

- 结构是否接近律师可用格式
- 是否能基于会议纪要生成
- 是否可被律师修改后交付
- 是否区分事实、分析、建议和风险提示

## 6. 总体验证问题

给律师测试时重点问：

1. 哪些内容你可以直接保留？
2. 哪些内容必须改？
3. 哪些内容有法律风险？
4. 是否节省了咨询后整理时间？
5. 你愿意继续给第二条真实咨询记录测试吗？
