# 待补材料清单

## 一、必须补充材料

| 材料 | 用途 | 说明 |
| --- | --- | --- |
| 劳动合同 | 证明劳动关系、岗位、合同期限、工资约定 | 请提供完整合同文本或清晰照片 |
| 工资流水 | 证明工资标准和工资支付情况 | 建议提供入职以来或至少近 12 个月流水 |
| 解除通知或解除沟通记录 | 证明解除时间、解除理由、解除方式 | 如没有书面通知，请提供微信、邮件、录音等沟通记录 |
| 绩效考核相关材料 | 判断公司以绩效不达标解除是否有依据 | 包括绩效制度、考核结果、绩效沟通、改进计划等 |
| 微信/邮件/工作软件沟通记录 | 证明工作安排、解除沟通、加班安排 | 请按时间顺序整理关键截图 |

## 二、建议补充材料

| 材料 | 用途 | 说明 |
| --- | --- | --- |
| 考勤记录 | 判断出勤和加班情况 | 如系统无法导出，可提供截图或其他记录 |
| 排班表/项目上线记录 | 辅助证明加班安排 | 尤其是领导要求活动上线、临时加班的记录 |
| 社保和个税记录 | 辅助证明劳动关系和工资水平 | 可从官方渠道下载 |
| 年休假记录 | 判断未休年假工资主张 | 包括公司年假系统、请假记录、审批记录 |
| 离职结算材料 | 判断工资、补偿或赔偿是否已结清 | 如有离职单、结算单、签收文件需提供 |

## 三、如无法提供，需补充说明

- 公司是否从未出具书面解除通知。
- 考勤记录无法导出的原因。
- 是否曾签署任何离职、解除、补偿、结算文件。
- 是否有录音、证人或其他材料可替代证明。

## 四、客户发送说明

```text
为便于律师进一步判断，请您优先补充以下材料：劳动合同、工资流水、解除通知或解除沟通记录、绩效考核相关材料、关键微信/邮件记录。若部分材料暂时无法提供，可以先说明原因，或提供替代材料。涉及个人隐私或无关人员信息的内容，可以先做适当遮盖。
```

## 五、律师内部提示

- 最影响违法解除判断的材料：绩效制度、绩效考核结果、解除通知、解除沟通过程。
- 最影响赔偿计算的材料：工资流水、劳动合同、离职时间、工资结构。
- 最影响加班费主张的材料：考勤记录、加班安排记录、工作成果提交记录。
- 如客户曾签署离职结算或补偿协议，需重点核验协议内容和签署背景。
