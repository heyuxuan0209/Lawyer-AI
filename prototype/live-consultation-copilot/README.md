# 实时谈案 Copilot 静态原型

版本：v0.7-live-static-prototype

## 打开方式

直接用浏览器打开：

```text
prototype/live-consultation-copilot/index.html
```

## 当前能力

- 劳动争议咨询示例
- 客户 / 律师分角色实时转写流
- 点击“开始咨询”后自动流式播放模拟转写，无需手动点击下一段
- 右侧同步更新：
  - 案情要点
  - 建议追问
  - 模拟法条卡
  - 方案草稿
- 点击“结束咨询”生成：
  - 咨询纪要
  - 材料清单
  - 方案摘要
  - 模拟法律意见书格式样例
- 点击右侧已生成产物按钮，可在中间区域查看内容预览

## 重要边界

法律意见书是 **模拟格式样例**，只用于让律师确认结构是否合适。

```text
未经律师审核，不可发送客户。
法条与结论均需进一步核验。
```
