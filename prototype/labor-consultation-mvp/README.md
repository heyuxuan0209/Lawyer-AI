# 劳动争议咨询助手静态 MVP

这是 `v0.3-static-prototype` 的静态原型，用于给律师朋友演示产品形态和收集反馈。

## 打开方式

直接用浏览器打开：

```text
prototype/labor-consultation-mvp/index.html
```

## 当前能力

- 三栏式工作台
- 左侧咨询线程列表
- 中间咨询处理流
- 右侧案情结构化、材料清单、交付件和律师确认
- 内置劳动争议示例咨询
- 支持粘贴咨询文本并生成模拟输出

## 暂未实现

- 真实 AI 生成
- 实时录音转写
- 法条检索
- 类案检索
- 数据持久化
- Word 导出

## 验证目标

拿给律师朋友时，重点验证：

1. 三栏式结构是否符合律师处理咨询的习惯。
2. 右侧案情结构化字段是否够用。
3. 交付件顺序是否合理。
4. 律师是否愿意把真实脱敏咨询记录粘贴进来试跑。
